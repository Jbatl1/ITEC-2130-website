Software Development

Software Development is the process of working with oter people to create and maintain a program for other people to use. This concentration focuses on teaching students the fundamental concepts of coding and then going more in depth into the actual software development process by using certain methodologies like waterfall or agile.

Technical skills

Java, Python, HTML, CSS, R, RStudio, Microsoft Excel, Microsoft Word, Microsoft Access , Microsoft Powerpoint, Microsoft Office.
